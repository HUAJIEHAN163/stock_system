# 🚀 股票分析系统安装指南

## 📋 系统要求

- **Python**: 3.8+ 
- **操作系统**: Windows/Linux/macOS
- **内存**: 建议4GB+
- **网络**: 需要访问tushare.pro或tudata

## 📦 快速安装

### 1. 安装Python依赖

```bash
# 安装所有依赖包
pip install -r requirements.txt

# 或者安装核心依赖
pip install PyQt5 pandas numpy tushare tudata requests beautifulsoup4
```

### 2. 配置Token

1. 获取Tushare Token: [https://tushare.pro/register](https://tushare.pro/register)
2. 或获取Tudata Token: [https://tudata.cn](https://tudata.cn)

### 3. 运行程序

```bash
# 方法1: 直接运行启动脚本
python run.py

# 方法2: 运行主程序
python src/main.py
```

## 🔧 首次使用

### 1. 设置Token

1. 启动程序后，点击"系统设置"
2. 在Token设置页面输入您的Token
3. 选择Token类型（tushare或tudata）
4. 点击"验证Token"确认可用性

### 2. 初始化数据

1. 在股票列表页面点击"数据初始化"
2. 等待程序下载基础股票数据
3. 初始化完成后即可使用所有功能

## ⚠️ 注意事项

### Token权限
- 不同Token有不同的API访问权限
- 部分高级功能需要更高积分等级
- 建议使用tudata获得更好的数据访问体验

### 数据更新
- 首次使用需要进行数据初始化
- 建议每日进行增量数据更新
- 可设置自动更新（功能开发中）

### 系统性能
- 大量数据查询可能需要较长时间
- 建议在网络良好时进行数据更新
- 可根据需要调整查询参数

## 🐛 常见问题

### Q: 提示"Token验证失败"？
A: 请检查Token是否正确，网络是否正常，账户积分是否足够

### Q: 数据初始化失败？
A: 检查网络连接，确认Token有效，重试初始化操作

### Q: 界面显示异常？
A: 确认PyQt5安装正确，尝试重启程序

### Q: 功能按钮无响应？
A: 部分功能仍在开发中，请查看项目进度文档
